# Sentinel — Surveillance Detection Dashboard

Brief
Sentinel helps you reduce your digital footprint and detect physical/algorithmic surveillance. This repository contains the dashboard and tooling to run locally or deploy.

Features
- Dashboard to visualize possible surveillance signals
- Tools to discover and catalog devices and cameras
- Privacy and hardening suggestions for users

Quick start (Node.js)
Prerequisites: Node.js (LTS recommended) and npm

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open your browser at:
   - http://localhost:3000 (or the port your app uses)

Available scripts
- `npm run dev` — run the development server
- `npm start` — start production server (if configured)
- `npm test` — run tests

Project structure (example)
- src/            — application code
- public/         — frontend / static assets
- tests/          — unit/integration tests
- docs/           — additional documentation
